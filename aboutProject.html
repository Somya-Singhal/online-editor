Online CODE EDITOR

Hello there!
Welcome to my Project->Online Code Editor.Let's take a quick overview of the project to know what's it all about.
This is an Online Code Editor for HTML, CSS, JS code snippets using only HTML, CSS, and React.As online code editors are fast, efficient, and greatly popular, it is a familiar tool among developers. So I was wondering, how it can be made? So, to get the liberty of using a local code editor, I started to implemented it.

Features

  1. Autocompletion
  2. Toggle Comment
  3. Searching and replacing words
  4. Jump to line
  5. Syntax Highlighting
  6. Displaying line numbers
  7. AutoSave Enabled
  8. Refresh option
  9. Toggle window
  10. Easy to use
